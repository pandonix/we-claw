这是一个新项目，准备实现一个基于 openclaw 的前端交互页面，类似 cowork 这样的交互方式，可安装在本地，但 agent 的内核直接使用 openclaw。

我会把 We-Claw 做成本地 Web + Node 工作台。Node 负责托管 OpenClaw Gateway，浏览器通过 Gateway WebSocket 驱动 session、chat、tool events 和 approvals。
